# 🎯 Azure AI Fundamentals

## Overview
This learning path covers the foundational concepts of Azure AI Services, including setup, security, deployment options, and best practices.

## Labs in this Path

### 1. Azure Setup & Configuration
- Creating Azure AI Services resources
- Understanding pricing tiers
- Regional deployment considerations
- Resource management

### 2. Security & Authentication
- API key management
- Azure AD authentication
- Network security
- Role-based access control (RBAC)

### 3. Container Deployment
- Running AI Services in containers
- Docker configuration
- Kubernetes deployment
- Edge deployment scenarios

## Prerequisites
- Azure subscription
- Basic understanding of cloud concepts
- Familiarity with REST APIs

## Getting Started
Begin with the Azure Setup lab to create your first AI Services resource.
